Distractinator 7000 (2023)

Project Status: COMPLETE, FUNCTIONAL

Goal: To create an attention tracking game using off-the-shelf parts and EEG (electroencephalogram) sensors.

Game Mechanics: The fan control module tries to distract the end-user by manipulating the fan speed to control the motion of white ping-pong balls based on user attention-levels (a number from 0 to 100) every 5 seconds. The game lasts 60 seconds. User with the highest average attention level at the end of the time period wins the game.

Link: Link to Presentation Video

Lessons Learned: Cost overruns must be mitigated by extensive preliminary research. Fan control system faced large voltage dips during operation. This caused unintended microcontroller restarts. The voltage dip issue was solved by adding an independent power supply.
